Memory Game

Welcome to Memory Game! Choose a grid, memorize the pictures inside, and match them with the same picture in other grids. Ready to test your memory?

How to Play

  • 1. Choose a grid size for your game.
  • 2. Memorize the pictures inside the grid for a few seconds.
  • 3. After the timer runs out, try to match the pictures from memory.
  • 4. Continue until all pairs are matched.
  • 5. Try to complete the game in the shortest time possible!
